## Installation

Install my-project with npm

```bash
  git clone https://github.com/matt-carmo/matt-carmo-desafio-maquina-de-sistemas
  cd desafio-maquina-sistemas-react
  npm install
  npm run dev
```
    
## Tech Stack

**Client:** React(tsx), Vite, TailwindCSS
